# practice_5
## API
```
https://github.com/theunclehonnor/vue_js_practice/tree/main/api
```
## Project setup
```
npm install
```
## Edit .env
```
Укажите ключ доступа VK API в файл `.env` 
VUE_APP_ACCESS_TOKEN_VK = Ваш ключ
```
### Project run
```
npm run serve
```
